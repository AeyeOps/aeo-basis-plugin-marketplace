# EFIN and Authorized E-File Provider

**Purpose**: IRS e-file provider authorization process, EFIN assignment, provider types, renewal, e-file mandate compliance, and advertising standards.

**Skills**: `tax-compliance`, `e-filing`, `provider-management`, `firm-administration`

---

## Overview

To electronically file tax returns with the IRS, a firm must become an **Authorized IRS e-File Provider** and obtain an **Electronic Filing Identification Number (EFIN)**. The application is submitted through the IRS e-Services portal. EFINs are issued per firm (not per individual preparer). Over 90% of all individual federal returns and a growing share of business returns are now e-filed.

## Application Process (3 Steps)

### Step 1: Create e-Services Account

- Register at the IRS e-Services portal
- Verify identity with SSN, prior-year tax return data, address, and contact info
- Account approval may take several business days

### Step 2: Submit Application

- Log in to e-Services and complete the e-file application (formerly Form 8633, now online-only)
- Required information:
  - Firm details (EIN, business name, address)
  - Provider type selection (ERO, transmitter, software developer, or intermediate service provider)
  - Principal(s) information — for partnerships: partners with 5%+ interest; for corporations: president, VP, secretary, treasurer
  - **Responsible Official** — individual with PTIN who oversees e-file operations and compliance
  - Primary contact information
  - Professional certification status (CPA, attorney, enrolled agent) if applicable
- All principals and the responsible official must have valid PTINs ($18.75 application fee)

### Step 3: Suitability Check

IRS reviews the firm and all listed individuals:
- **Credit check** — Financial responsibility assessment
- **Tax compliance check** — All personal and business tax obligations must be current
- **Criminal background check** — May include fingerprinting (see below)
- **Prior e-file compliance check** — History of sanctions or non-compliance reviewed
- Processing time: **up to 45 days** from submission

**Fingerprinting**: Individuals without professional licenses (CPA, attorney, EA) must complete Livescan electronic fingerprinting through IRS-authorized vendors. Appointments are scheduled via the e-file application summary page. Vendors are available nationwide within a 120-mile radius of the applicant, covering all 50 states, D.C., Guam, Puerto Rico, Saipan, and USVI.

### Approval

Upon approval, IRS issues an acceptance letter containing the firm's EFIN. Denial may result from unpaid taxes, financial felonies, fraud history, or prior e-file non-compliance. Denied applicants may appeal or reapply after correcting deficiencies.

## Provider Types

### Electronic Return Originator (ERO)

- **Most common type** for tax preparation firms
- Prepares, collects, and originates returns for electronic filing
- Responsible for obtaining taxpayer signatures (Form 8879 series or Form 8453 series)
- Must verify all Forms W-2, W-2G, and 1099-R are received from taxpayer before submitting
- ERO is the provider type to select for firms that prepare and file client returns

### Transmitter

- Sends electronic return data to IRS
- May or may not also be an ERO
- Must pass ATS (Assurance Testing System) certification
- Operates the communication link with IRS MeF system

### Software Developer

- Develops tax preparation software that generates MeF-compliant XML
- Must pass ATS certification for each supported form type
- Software is listed on IRS approved provider directory upon certification

### Intermediate Service Provider (ISP)

- Assists in the electronic filing process between ERO and transmitter
- Receives returns from EROs and forwards to transmitters
- Firm-based EFIN required

## Responsible Official

- Individual designated to oversee the firm's e-file operations
- Must have a valid PTIN
- Must pass suitability check (same as principals)
- Accountable for firm compliance with all e-file rules
- Must update information within **30 days** of any changes
- Each firm must designate at least one responsible official

## Annual Renewal

- EFINs require **annual renewal** through e-Services to maintain authorization
- Monitor EFIN status proactively; failure to renew results in deactivation
- Renewal involves confirming firm details, principals, and responsible official remain current
- No fee for renewal

## E-File Mandate for Tax Return Preparers

Any tax return preparer who files **10 or more returns** (aggregated across all return types: income, employment, excise, information) during a calendar year is required to e-file. This replaced the prior 250-return threshold.

### Threshold Changes (Effective Date Log)

- **Pre-2011** — 250 returns threshold for information returns
- **2011-2023** — 250 returns threshold maintained
- **2024-01-01** — Threshold reduced to **10 returns** (T.D. 9972, implementing Taxpayer First Act Section 2301, IRC 6011(e)(3))
- Returns are aggregated across types — a firm filing 5 W-2s and 5 1099s meets the threshold
- Penalty for failure to e-file when required: up to **$310 per return** (IRC Section 6721, inflation-adjusted for TY 2024+)
- Hardship waivers available via **Form 8508**

## PTIN Requirements

- All paid tax return preparers must have a valid **Preparer Tax Identification Number (PTIN)**
- Required for both the responsible official and any individual who prepares returns
- Application fee: $18.75 (subject to change)
- Requires SSN, personal/business information, and prior-year return data
- Must be renewed annually

## Advertising Standards

IRS-authorized e-file providers must follow strict advertising rules (Publication 3112, Rev. Proc. 2007-40):

**Prohibited**:
- Using "Internal Revenue Service" or "IRS" in firm name or advertising
- Claiming guaranteed refund timing (e.g., "refund in 24 hours")
- Using IRS logo, eagle symbol, IRS insignia, or Treasury seals
- Implying a special relationship with IRS

**Permitted**:
- Using the "IRS e-file" logo (without endorsing elements like "federal")

**Record retention**: If ads include fee information (radio, TV, internet, signage), retain copies for **36 months** from last use/transmission and provide to IRS on request.

Non-compliant advertising triggers monitoring: providers receive warnings to correct, escalating to sanctions for persistent violations.

## Sanctions

IRS sanctions for e-file providers follow a tiered severity model:

- **Level 1** — Written reprimand
- **Level 2** — Suspension from e-file program
- **Level 3** — Expulsion from e-file program

Triggers include: advertising violations, suitability failures, unauthorized disclosure of taxpayer data, filing returns without proper authorization, failure to comply with Rev. Proc. 2007-40 or Publications 3112/1345.

The IRS e-file Monitoring Coordinator handles escalated cases.

## Record Retention Requirements

- **Form 8879 series** (e-file signature authorization) — 3 years from return due date or IRS receipt date, whichever is later
- **Form 8453 series** — 3 years
- **Advertising with fees** — 36 months from last use
- **Electronic return data** — 3 years (per Rev. Proc. 2007-40)
- **Copy of filed return** — Varies; follow IRC Section 6107 (3 years for preparers)

## Decision Points

1. **Which provider type?** Most tax firms select ERO. Choose transmitter only if operating the MeF communication channel. Software developer if building tax software
2. **Who is the responsible official?** Must have PTIN, pass suitability, and actively oversee compliance
3. **Fingerprinting needed?** Only for individuals without CPA, attorney, or EA credentials
4. **Does the firm meet the e-file mandate threshold?** Count all returns (income + employment + excise + information) filed during the calendar year. If 10+, e-filing is required for TY 2024+
5. **EFIN renewal current?** Check e-Services annually; deactivation prevents filing
